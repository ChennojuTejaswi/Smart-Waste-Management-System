# Smart Waste Management System

This project is a web-based application that classifies waste using Deep Learning and helps users identify recyclable materials and nearby waste centers.

## Features
- Waste image classification using MobileNet (CNN)
- User login and upload functionality
- Admin dashboard to manage waste centers
- Displays classification results

## Technologies Used
- Python
- Flask
- TensorFlow / Keras
- HTML, CSS, JavaScript
- SQLite

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/

## Note
Model file (.h5) is not included due to size limitations.
